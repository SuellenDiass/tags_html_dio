Definição do HTML

A Linguagem de Marcação de Hipertexto(HTML) é uma linguagem de marcação usada para criar sites. Ela utiliza tags HTML para estruturar páginas da web, permitindo que tenham cabeçalho, corpo, barra lateral e rodapé.

Tags html também são usadas para formatar texto, incorporar imagens ou atributos, criar listas e ligar arquivos externos. Essa última função permite que você modifique o visual da sua página ao incorporar arquivos CSS* e outros objetos.

É importante usar as tags HTML adequadas, já que a estrutura incorreta pode quebrar a página da web. Pior do que isso, os motores de busca podem não serem capazes de ler a informação apresentada nas tags.

Abaixo segue as guias de referências com as tags mais utlizadas no HTML e também as menos utilizadas. Aprendemos que é possível criar uma tabela e que podemos fazer algumas estilizações usando uma folha de estilo (style).

Estrutura do HTML

(Voltar) 
<!DOCTYPE html>
     <html>
     <head>
     <title>Título da página</title>
     <meta charset="utf-8"/>
     </head>
     <body>

     </body>
     </html>

Tag <html> mostra que a página usa o html

Tag <head> essa tag delimita o cabeçalho do documento. Não possui nenhum valor visível, porém é capaz de transmitir ao navegador diversas informações como nome da página,uso de fonte,e suas dependências

Tag <title>essa tag define o título da sua página, o nome que aparecerá na sua aba, janela ou guia. Por esta razão, a tag >title> é de grande importância para o SEO;

Tag <meta charset="utf-8"/> essa tag permite inserir metadados ao seu documento, no caso acima, a informação charset=”UTF-8″, que garante a compatibilidade do código com os caracteres de padrão latino americano;

Tag<body> corpo do documento, tudo que o usuário vê é escrito dentro desta tag
